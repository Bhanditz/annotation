Europeana Creative - Annotation Component Management Console

Management console for annotation component is a service that implements the management of database and SOLR entries for Web Annotations

Delete Annotations API

Method Response Path Function
GET TEXT [/annotation-web]/annotations/admin/component Display the name of the current component
POST JSON [annotation-web]/annotations/admin/delete
Delete annotation by ID: 
annotation Nr: 
europeana ID: 
 Delete the annotation identified by the given ID from database and indexing.
POST JSON [annotation-web]/annotations/admin/index
Index annotation by ID: 
annotation Nr: 
europeana ID: 
 Index the annotation identified by the given ID. This service forces reindexing of the annotation, which means deletion in SOLR annotation if exists and creation of new entry in SOLR annotation.
POST JSON [annotation-web]/annotations/admin/disable
Disable annotation by ID: 
annotation Nr: 
europeana ID: 
 Disable the annotation identified by the given ID. This service sets field 'disabled' in annotation object to true in database and removes the annotation from the SOLR annotation.
