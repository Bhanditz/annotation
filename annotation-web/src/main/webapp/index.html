Europeana Creative - Annotation Component

Annotation component is a service that implements the management and indexing of UserTags and ImageAnnotations
Method Response Path Function
GET TEXT [/annotation-web]/annotations/component Display the name of the current component
GET JSON [/annotation-web]/annotations/{collection}>/{object}?wskey=&profile= get the list of annotations available for the given object
GET JSON [annotation-web]/annotations/{collection}/{object_hash}/{annotationNr}?wskey=&profile= get the annotation identified by the given annotation NR
POST JSON [annotation-web]/annotations/{collection}/{object_hash}
ObjectTag as JSON:

create a semantic tag (BodyType) and return the object as stored in the database
POST JSON [annotation-web]/annotations/{collection}/{object_hash}
ObjectTag as JSON:

create a simple tag (BodyType) and return the object as stored in the database
POST JSON [annotation-web]/annotations/{collection}/{object_hash}
ObjectTag as JSON:

create an (simple) image annotation (BodyType) and return the object as stored in the database
